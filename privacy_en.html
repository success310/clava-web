2. DATA CATEGORIES, PURPOSES AND LEGAL BASIS

2.1. Connection setup

Every time a user accesses, the device used automatically transmits the following data to the Clava Sports web server for technical reasons:
  • IP address
  • Date and time of access
  • Name and URL of retrieved file
  • Amount of data transferred
  • Identification data of the device used and its operating system
  • Name of the provider of the mobile or WiFi network of the user
  • Username (when the visitor logs in with a username and password)
  • Browser type/ version
  • browser language
  • Selected region
  • Favorite Teams
  • Favorite Leagues
  • Inner resolution of the browser window
  • Screen resolution
  • Javascript activation
  • Cookies On / Off
  • Color Depth
  • Referrer URL (of the previously visited page/s)
  • Clicks
This data is processed for the purpose of enabling the use of the app (establishing a connection), system security and technical administration and improving the network infrastructure. A comparison with other databases or a transfer to third parties, even in excerpts, only takes place in an anonymous form. In addition, it is used anonymously to collect statistics. For more information on web analysis, see Section 4 "Web Analysis" below. The legal basis for data processing is Art. 6 Para. 1 S. 1 lit. f GDPR. Our legitimate interest follows from the data collection purposes listed above. Under no circumstances do we use the collected data for the purpose of drawing conclusions about your person. We delete this data after 50 days.

2.2. Clava Account

In order to create a Clava account, each user must register for the Clava Sports digital offering. The following data is collected for this purpose:
  • First and last name
  • Email address
  • Username
  • Password
  • phone number
We need this information in order to be able to allocate an account to a specific user.
A logged-in user has the option of running a fanticker. The username is published here.
After registering and logging in for the first time, the current terms of use are automatically displayed, the current version of which can be accessed at https://www .clava-sports.com/tos.html.

2.3. Player Profile

Registered users can take over a registered player profile. Here we offer the option of voluntarily providing the following data:
  • Profile picture
  • Age
  • Birthday
  • Location
  • Size
  • Preferred Foot
Providing this additional information can make the Clava Account more attractive and meaningful. This should enable stronger networking and interaction between users and strengthen the sense of community in the South Tyrolean football community as a whole.
The data collected in this context will be processed to create and display the account. The legal basis for processing is Art. 6 (1) (b) GDPR or Art. 6 (1) (a) GDPR if you provide data voluntarily.

3. RIGHTS OF THE DATA SUBJECT

If your personal data is processed, you have the right to receive information about the data stored about you (Article 15 GDPR).
Should incorrect personal data be processed, you have the right to rectification (Article 16 GDPR).
If the legal requirements are met, you can request the deletion or restriction of processing and object to the processing (Art. 17, 18 and 21 DSGVO).
If you have consented to the data processing or if there is a data processing contract and the data processing is carried out using automated procedures, you may have a right to data portability (Article 20 GDPR).
To exercise your rights, please contact privacy@clava.sports.com.
If you have any complaints about data protection law, you can contact the responsible supervisory authority.

9. STORAGE DURATION

Personal data is processed and stored for as long as the purpose for which it was collected requires. Therefore:
  • Personal data collected for the purpose of fulfilling a contract concluded between the provider and the user will be stored until the contract has been completely fulfilled.
  • Personal data collected to safeguard the legitimate interests of the provider will be retained for as long as is necessary to fulfill these purposes. Users can obtain more detailed information about the legitimate interests of the provider in the relevant sections of this document or by contacting the provider.
In addition, the provider is permitted to store personal data for a longer period of time if the user has consented to such processing, as long as the consent is not revoked. In addition, the provider may be obliged to store personal data for a longer period of time if this is necessary to fulfill a legal obligation or by order of an authority. Once the retention period has expired, personal data will be deleted.
